# github_upload.py Teknik Notu

## Rolü
- Yerel değişiklikleri adım adım commit/push akışıyla GitHub’a göndermeyi kolaylaştıran yardımcı CLI scriptidir.

## Öne Çıkan Noktalar
- Remote tespiti, commit mesajı alma, push ve bazı conflict senaryoları için otomatik akış içerir.
- Kullanıcıya renkli terminal çıktısı ile rehberlik eder.

## İyileştirme Alanı
- Rebase/merge stratejileri ve güvenlik kontrolleri (secret scanning ön kontrolü) daha deterministik hale getirilebilir.
